## 查看项目文件

在上一课中，我们在 Sui 上生成了一个名为 "move_castle" 的初始 Move 项目。现在，让我们来探索项目的内容。

目录结构清晰简单：

```
move_castle
├── Move.toml
├── sources
│   └── move_castle.move
└── tests
    └── move_castle_tests.move
```

项目文件夹的结构如下：

1. 项目根文件夹，名为 `move_castle`
2. 清单文件 `Move.toml`
3. 包含 .move 合约文件的 `sources` 子文件夹
4. 包含测试文件的 `tests` 子文件夹
